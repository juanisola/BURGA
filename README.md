# 🍔 BURGA

Sitio web para una hamburguesería.  
Presenta el menú, la identidad del local y la información básica para clientes.

---

## 📌 Descripción

BURGA es una página web pensada para mostrar de forma clara:

- Productos (hamburguesas, combos, etc.)
- Información del local
- Contacto y ubicación

El enfoque está en una interfaz simple, visual y fácil de navegar.

---

## 🛠️ Tecnologías

- HTML5  
- CSS3  
- Flexbox / Grid  

---


---

## 🎯 Objetivo

Desarrollar una web funcional y clara para una hamburguesería, priorizando:

- Buena organización del contenido  
- Diseño limpio
